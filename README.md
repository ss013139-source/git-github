Boa noite a todos,

Hoje concluo o curso de Git e GitHub oferecido pela plataforma Estudonauta, ministrado pelo professor Gustavo Guanabara. Ao longo dessa jornada, tive a oportunidade de compreender não apenas os comandos e funcionalidades dessas ferramentas, mas principalmente a importância do controle de versão no desenvolvimento de software.

Durante o curso, aprendi conceitos fundamentais como versionamento, repositórios, commits, branches e merges, entendendo como o Git atua no gerenciamento do histórico de alterações de um projeto. Também aprendi a utilizar o GitHub como plataforma de hospedagem de código, colaboração em equipe e compartilhamento de projetos.

Um dos pontos mais importantes foi perceber como o uso correto do Git proporciona organização, segurança e produtividade, permitindo que desenvolvedores trabalhem de forma colaborativa, mantendo o controle sobre cada etapa do desenvolvimento.

A didática do professor Gustavo Guanabara tornou o aprendizado leve, prático e acessível, facilitando a compreensão mesmo dos conceitos mais técnicos. Os exemplos práticos e exercícios propostos contribuíram significativamente para fixar o conteúdo.

Concluir este curso representa um passo importante na minha formação na área de tecnologia, pois o domínio de Git e GitHub é essencial para qualquer desenvolvedor que deseja atuar profissionalmente no mercado.

Agradeço à plataforma Estudonauta e ao professor Gustavo Guanabara pelo conteúdo de qualidade e pela oportunidade de aprendizado.

Muito obrigado!
